# Blacksword Studio

The official website of **Blacksword Studio** — an independent one-person game studio dedicated to crafting original games and custom software. Built with [Astro](https://astro.build) and deployed via GitHub Pages.

🌐 **Live Site**: [blackswordstudio.github.io](https://blackswordstudio.github.io/)

## 🎮 About

Blacksword Studio is a solo indie game development studio. Without the constraints of large teams or corporate processes, every aspect of development — from gameplay concepts and code to level design and visual polish — is crafted with a singular creative vision.

## 🛠️ Tech Stack

- **[Astro 5](https://astro.build)** — Static site generator
- **GitHub Pages** — Hosting & deployment
- **GitHub Actions** — CI/CD (auto-deploy on push to `master`)
- **i18n** — English (`en-US`) & Simplified Chinese (`zh-CN`)
